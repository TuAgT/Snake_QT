# 贪吃蛇游戏

使用 Qt C++ 框架实现的经典贪吃蛇游戏。

## 功能特性

- 经典贪吃蛇玩法
- 分数跟踪
- 键盘控制
- 暂停/恢复功能
- 带游戏状态显示的精美UI

## 控制方式

- 方向键或 WASD: 控制蛇的移动
- 空格键: 暂停/恢复游戏
- R键: 重新开始游戏
- Ctrl+N: 开始新游戏
- Ctrl+P: 暂停/恢复游戏
- Ctrl+Q: 退出游戏

## 构建项目

此项目使用 CMake 作为构建系统。构建和运行方法如下：

```bash
mkdir build
cd build
cmake ..
make
./QTGuiDemo
```

## 项目结构

- `main.cpp`: 应用程序入口点
- `GameWindow.h/cpp`: 带菜单和状态显示的主窗口
- `SnakeGame.h/cpp`: 游戏逻辑（蛇的移动、碰撞检测、食物生成）
- `SnakeWidget.h/cpp`: 游戏渲染和用户输入处理

## 架构设计

游戏遵循 Model-View-Controller (模型-视图-控制器) 模式：

- **模型 (`SnakeGame`)**: 处理所有游戏逻辑，包括蛇的位置、食物放置、碰撞检测和计分
- **视图 (`SnakeWidget`)**: 负责渲染游戏图形和处理用户输入
- **控制器 (`GameWindow`)**: 提供整体UI结构并整合所有组件
